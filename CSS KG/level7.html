Animation , Transition & Transform

Pseudo Classes

used to define special states of html elements

hover, active, focus, visited, link

  • hover: when mouse is over the element
  • active: when element is being clicked
  • focus: when element is focused
  • visited: when link has been visited
  • link: when link is not visited

    • Transition

    used to change property values smoothly (over a given duration)

    this enables smooth animation changes

  • transition-property: css property to be transitioned
  • transition-duration: duration of the transition
  • transition-timing-function: speed curve of the transition
  • transition-delay: delay before transition starts
    • transition: "all 2s ease-in-out 1s;"
    syntax for all properties in one single line

    Animation

    used to create more complex animations

    keyframes are used to define the animation sequence

    animation properties are used to define the animation

  • animation-name: name of the keyframe
  • animation-duration: duration of the animation
  • animation-timing-function: speed curve of the animation
  • animation-delay: delay before animation starts
  • animation-iteration-count: number of times the animation should play
  • animation-direction: direction of the animation

    • Animation

    " animation: name duration timing-function delay iteration-count direction fill-mode; "
    syntax for all properties in one single line

    Using percentages in animations allows you to define the state of an element at specific points during the animation.
