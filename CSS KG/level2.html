Height & Width property

height and width property is used to set the height and width of an element

can use units like px , % , em ,rem , vh , vw , vmin , vmax , auto

box 1

height: 100px; width: 100px;

can us height and width property to set the height and width of any of the element

Background Image Property

background image property is used to set the background image of an element

background image can be set using url() function

background size property is used to set the size of the background image

background repeat property is used to set the repeatition of the background image

background position property is used to set the position of the background image

background attachment property is used to set the attachment of the background image

background origin property is used to set the origin of the background image

Visibility property

visibility property is used to set the visibility of an element

visibility property can have two values visible and hidden

visible - element is visible

hidden - element is hidden

can use visibility property to hide any of the element

above box is hidden
below box is visible

if needed you can inspect the element to see the hidden box
