BOX MODEL

Consists of four main components

  1. Content
  2. Padding
  3. Border
  4. Margin

Each of these components are crucial in creating a layout for a webpage

Content: This is the area where the content of the element is displayed, which can be text, an image, a video, etc.

Padding: This is the area between the content and the border of the element. It is used to create space around the content.

Border: This is the area that goes around the padding and content of the element. It is used to define the boundary of the element.

Margin: This is the area outside the border of the element. It is used to create space between the element and other elements on the page.


Padding Property

The padding property is used to generate space around an element's content, inside of the element's border. The padding is affected by the background color of the element.

Padding has four sides, which can be set independently:

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

Margin Property

The margin property is used to generate space around an element, outside of the element's border. The margin is affected by the background color of the element.

Margin has four sides, which can be set independently:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

Border Property

The border property is used to set the width, style, and color of an element's border.

The border property has three parts:

  • border-width
  • border-style
  • border-color

The border property can be set for all four sides of an element, or for each side individually:

  • border-top
  • border-right
  • border-bottom
  • border-left

border raidus

The border-radius property is used to add rounded corners to an element.

Four values can be specified to define the radius of each corner

border-radius: 25px 25px 30px 45px;

Box Sizing

The box-sizing property is used to tell the browser what the sizing properties (width and height) should include.

By default, the width and height properties include the content, but not the padding, border, or margin.

However, the box-sizing property can be used to include the padding and border in the element's total width and height.

box-sizing: border-box;

box-sizing: content-box;
