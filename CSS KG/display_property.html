Display property Block / Inline elements

Block elements always start on a new line and take up the full width available

examples - div tag , p tag, h tags ,ul tag ,li tag

Inline elements only take up as much width as necessary and do not start on a new line, cannot set size easily

example- span tag , button

Display property values

block- the element will be displayed as a block-level element, with a line break before and after the element

inline- the element will be displayed as an inline-level element, with no line break before or after the element

inline-block- the element is displayed as an inline-level block container

none- the element is completely removed
